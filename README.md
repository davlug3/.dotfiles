# My Dotfiles repo

*this is primarily used for vim, but i may use this for other things as well*

Instructions

```
    # clone the repo
    cd ~
    git clone https://github.com/davlug3/.dotfiles

    # install vundle 
    ./.dotfiles/install_vundle.sh

    # create the symlinks using stow
    ./.dotfiles/install.sh
```
